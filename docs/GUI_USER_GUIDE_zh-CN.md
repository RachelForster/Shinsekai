# 新世界程序 · 图形界面使用指南（新手向）

如果您不熟代码、也不想碰命令行，可以只照着本指南在屏幕上点选完成配置与游玩。界面以 **中文** 为例；若您把软件语言切到英文，按钮名字会对应变化，但位置相同。

---

## 1. 您会用到两个不同的窗口

| 窗口 | 像什么 | 干什么 |
|------|--------|--------|
| **设置窗口** | 手机的「系统设置」 | 填 AI 账号、管理角色与立绘、生成聊天规则、装插件。标题里通常带有 **「设置」** 字样。 |
| **聊天窗口** | 游戏的「对话界面」 | 和角色打字聊天、看立绘换人、听语音（若已开启）。从这里专注「演戏」，少改复杂选项。 |

**常用顺序：** 先在 **设置** 里把 API 和人物准备好 → 到 **聊天模板** 点 **启动聊天** → 弹出 **聊天窗口** 再开始玩。

---

## 2. 怎样打开设置窗口

- **下载了整合包：** 解压后双击 **`start.bat`**（有的版本名称接近，以您包内说明为准）。  
- **从源码安装的玩家：** 在程序所在文件夹里用说明里写的方式运行 **`webui.py`**（若不会操作，可优先使用整合包，一般只需双击）。

打开后，您应能看到 **左侧一列竖排按钮** 和 **右侧大块的设置内容**。

---

## 3. 左侧菜单分别管什么（从上到下）

点左侧按钮即可切换页面，不必记快捷键。

| 按钮文字（中文） | 您能在这里做什么 |
|------------------|------------------|
| **API 设定** | 配置「大脑」（大语言模型）、配音（语音合成）、麦克风识别（语音识别）、画画（文生图）等。**多数新手第一次必须先进这一页。** |
| **人物设定** | 导入/新建角色、编辑名字与简介、上传立绘、给每张图标情绪、给某张立绘挂台词音频等。 |
| **背景管理** | 导入或整理 **背景图组**（与别人分享时可能是 `.bg` 文件）。 |
| **聊天模板** | 勾选本次要出场的角色、选一些演出选项，让程序**自动生成**给 AI 看的「规矩说明书」；可 **保存** 为文件，并 **启动聊天**。 |
| **插件** | **管理插件**（开/关、卸载）、**发现插件**（从网络列表安装）、以及高阶的 **MCP**（连外部工具）。一般新手可稍后再看。 |
| **小工具** | 辅助小功能（具体以您安装版本为准）。 |

页面里有许多 **可折叠区块**（左边常有小三角 ▶）。点一下可 **展开/收起**，避免一次看到太多字。

---

## 4. API 设定（最重要，建议第一次就完成）

### 4.1 先搞懂三件事

1. **大语言模型（LLM）**：真正「替角色回话」的云端或在线 AI，需要 **API Key**（相当于账号密码里的一把钥匙）。  
2. **语音合成（TTS）**：要不要让程序把台词 **念出来**。可以以后再开。  
3. **语音识别（ASR）**：用麦克风说话时，把声音转成文字。默认一般是 **Vosk**；换别的可能要装插件。

文生图、ComfyUI 等属于 **进阶玩法**，不会用可以先全部空着或保持默认。

### 4.2 配置大语言模型（必做，才能聊天）

1. 左侧点 **「API 设定」**。  
2. 找到 **选择大语言模型供应商**（或类似下拉框），选一个您打算用的服务（例如说明里写的 DeepSeek 等）。  
3. 填写 **模型 ID**（模型名字，按服务商文档抄写即可）。  
4. 填写 **API Key**（从该服务商网站注册后生成，**不要发给别人**）。  
5. 填写 **API 基础网址**（有的选项选完后会自动填，一般不用改）。  
6. 拉到本页 **最底部**，点击 **保存** 类按钮（鼠标悬停时，提示里会说明会保存到配置文件）。

**保存成功后才算数**；若关掉窗口前没保存，刚才填的不会生效。

### 4.3 配音（TTS）与麦克风（ASR）

- **TTS：** 在 **API 设定** 里选择 **语音引擎**（例如不用配音、或 GPT-SoVITS、Genie TTS 等）。若选需要本地服务的引擎，说明里会提示您下载整合包，并把 **路径或网址** 填到对应格子。  
- **ASR：** 同一页或相关区域可选择 **麦克风识别** 用的后端；新手保留默认即可。若选了某项后多出**额外输入框**，按插件或教程说明填写。

### 4.4 文生图（可跳过）

若您暂时不让 AI 在对话里「出图」，可以先不填 ComfyUI 地址与工作流。需要时按软件内**资源说明**或视频教程一步步来。

---

## 5. 人物设定（角色与立绘）

1. 左侧点 **「人物设定」**。  
2. **导入别人做好的角色包：**  
   - 点 **选择文件** / **从文件导入**，选一个或多个 **`.char`** 文件；  
   - 再点 **从文件导入人物**（或类似按钮），等待完成提示。  
3. **自己改名字或剧情：** 在 **人物名称**、**角色设定** 等大文本框里编辑；可用 **AI 一键帮写**（需已在 **API 设定** 里配好模型）。  
4. **立绘：** 点 **选择立绘图片** / **上传图片**，设 **立绘显示缩放**，在情绪栏给每张图写 **关键字**（方便对白里自动换表情）。  
5. 改完后在 **保存人物档案** 区域点 **添加或保存人物设置**，避免关掉后丢失。

---

## 6. 背景管理

- 用来管理多套 **场景背景**（可能是 **`.bg`** 文件）。  
- **导入：** 点 **选择文件…** → **从文件导入背景组**。  
- 编辑名称、预览图后同样记得在页面里找 **保存** 类操作。

---

## 7. 聊天模板（生成规则 → 启动聊天）

1. 左侧点 **「聊天模板」**。  
2. 在 **生成模板（选人物与选项）** 区域 **勾选** 本次要出场的角色（至少一个）。  
3. 按需调整 **系统生成模板**、**模板规则** 等选项（新手可先保持默认，再慢慢试。页内 **情景** 可手写剧情走向）。  
4. 点击 **生成** 类按钮（页面在改动部分选项时也可能自动生成），等待 **系统生成模板** 大框里出现文字（这是给 AI 的「演戏规则」，不必逐字背）。  
5. 在 **保存模板** 里起个文件名，点 **保存模板**，以便下次接着用。  
6. 点 **启动聊天**：会打开 **聊天窗口**。若提示找不到模板，请先完成上面 **生成** 与 **保存** 至少一次。

**小贴士：** 若主界面提供 **加载上次聊天并启动** 类按钮，可用最近的存档快速继续玩（以窗口上提示为准）。

---

## 8. 插件

- **管理插件：** 看到一张张卡片，可 **启用/禁用**；修改清单后一般要 **关掉设置程序再重新打开** 才加载完整。  
- **发现插件：** 从列表里 **下载** 扩展（需能上网）；按屏幕提示操作后同样建议 **重启** 程序。  
- **MCP：** 给高级用户连接外部工具用；若提示安装额外软件或失败，可先跳过，或阅读项目 [`README.md`](../README.md) 里 MCP 小节。

---

## 9. 聊天窗口里可以怎么玩（设置好之后）

- **输入框** 里打字发送，角色会按模板规则回复。  
- 若模板里带有 **选项**，可用鼠标点选分支。  
- 若开了 **语音合成**，且 API 里引擎配置正确，可能会听到台词。**没声音时**，回到 **API 设定** 检查 TTS 是否选成「不使用」或路径、服务是否未启动。  
- 界面上若有 **历史、清屏、跳过朗读** 等按钮，以当前界面文字为准。

## 10. 常见问题（小白自查）

| 现象 | 建议 |
|------|------|
| 点启动聊天提示没有模板 | 去 **聊天模板** 先 **生成** 并 **保存模板** 一次。 |
| 角色不说话 | **API 设定** 里语音引擎是否关闭；若用 SoVITS、Genie 等，目录或网址是否填对、对应程序是否在本机已打开。 |
| 装了插件没效果 | **管理插件** 里是否启用；是否已 **完全退出并重新打开** 设置程序。 |
| 改了 API 但不生效 | **API 设定** 页底部是否点了 **保存**。 |
| 完全不知道从哪开始 | **API 设定**（模型 + Key）→ **人物设定**（导入 `.char`）→ **聊天模板**（勾角色 → 生成 → 保存 → **启动聊天**）。 |
