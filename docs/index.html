安装与运行

  1. 下载上方最新版源码并解压，或 git clone https://github.com/RachelForster/Shinsekai
  2. 创建环境（推荐 Python 3.10）： 
    conda create -n shinsekai python=3.10
conda activate shinsekai
pip install -r requirements.txt
  3. 启动设置界面：
    python webui_qt.py
  4. 配置 API — 在 API 设定 中填写 LLM 供应商、Key 与模型
  5. 导入角色 — 角色管理 中导入 .char 角色包
  6. 启动聊天 — 点击按钮，开始对话

核心能力

  • 多 LLM 支持 — DeepSeek / OpenAI / Claude / Gemini 等
  • TTS 语音合成 — GPT-SoVITS / Genie TTS / CosyVoice
  • ASR 语音识别 — Vosk / Faster Whisper / RealtimeSTT
  • 文生图 — ComfyUI / Stable Diffusion
  • 插件系统 — 扩展适配器、工具、UI 组件
  • MCP 协议 — 接入外部工具与检索服务
  • 立绘情绪联动 · 聊天历史回溯 · BGM / CG 播放
