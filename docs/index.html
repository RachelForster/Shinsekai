快速开始

  1. 下载对应系统版本（上方 Release 按钮或镜像任选）：

    百度网盘 夸克网盘 Google Drive

  2. 打开下载的安装包或可执行程序：Windows 运行 Shinsekai.exe / .msi，macOS 打开 .dmg，Linux 运行 .AppImage 或安装 .deb
  3. 启动 Shinsekai 后，在 API 设定 填写 LLM 供应商、模型 ID、API Key 与 Base URL，保存
  4. 如果需要角色说话，在 API 设定 的 TTS 整合包 下载语音引擎（Genie TTS / GPT-SoVITS）
  5. 在 角色管理 导入角色包（更多见 社区资源），打开 聊天模板 生成模板并 启动聊天

🔄 更新软件：设置页面 → 插件 → 发现插件 → 更新软件，即可升级到最新版

📖 详细图文教程：新世界程序使用指南（语雀）

开发者安装（源码）

  1. 下载 GitHub 源码包（zip / tar.gz）并解压，或 git clone https://github.com/RachelForster/Shinsekai
  2. 创建项目标准环境（推荐使用 environment.yml）： 
    conda env create -f environment.yml
conda activate shinsekai
  3. 构建 React 前端： 
    cd frontend
pnpm install
pnpm build
cd ..
  4. 优先运行 python webui_react.py 或平台对应的 start-react.*；旧 PySide 设置页仅作为兼容入口保留：python webui_qt.py
  5. React 设置中心用于管理 API、角色、背景、聊天模板、小工具、插件和 MCP
  6. 完成配置后按上面的「第一次对话」流程启动聊天

核心能力

  • 多 LLM 支持 — DeepSeek / OpenAI / Claude / Gemini 等
  • TTS 语音合成 — GPT-SoVITS / Genie TTS
  • ASR 语音识别 — Vosk / Faster Whisper / RealtimeSTT
  • 文生图 — ComfyUI / Stable Diffusion
  • 插件系统 — 扩展适配器、工具、UI 组件
  • MCP 协议 — 接入外部工具与检索服务
  • 立绘情绪联动 · 聊天历史回溯 · BGM / CG 播放
