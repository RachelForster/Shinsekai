快速上手（整合包 · 无需技术背景）

  1. 下载整合包（任选其一）：

    百度网盘 夸克网盘 Google Drive

  2. 解压下载的文件，注意路径里不要有中文（比如直接放在 D 盘根目录）
  3. 双击 Install.bat 等待安装完成
  4. 双击 start.bat 打开设置页面
  5. 在 API 设定 中配置 LLM 供应商和 Key，保存
  6. 在 角色管理 中导入角色包（.char 文件）
  7. 打开 聊天模板，选择角色，生成模板
  8. 点击 启动聊天，开始对话

🔄 更新软件：设置页面 → 插件 → 发现插件 → 更新软件，即可升级到最新版

开发者安装（源码）

  1. 下载上方最新版源码并解压，或 git clone https://github.com/RachelForster/Shinsekai
  2. 创建环境（推荐 Python 3.10）： 
    conda create -n shinsekai python=3.10
conda activate shinsekai
pip install -r requirements.txt
  3. 启动设置界面： 
    python webui_qt.py
  4. 在 API 设定 中填写 LLM 供应商、Key 与模型
  5. 在 角色管理 中导入 .char 角色包
  6. 打开 聊天模板，选择角色，生成模板，启动聊天

核心能力

  • 多 LLM 支持 — DeepSeek / OpenAI / Claude / Gemini 等
  • TTS 语音合成 — GPT-SoVITS / Genie TTS
  • ASR 语音识别 — Vosk / Faster Whisper / RealtimeSTT
  • 文生图 — ComfyUI / Stable Diffusion
  • 插件系统 — 扩展适配器、工具、UI 组件
  • MCP 协议 — 接入外部工具与检索服务
  • 立绘情绪联动 · 聊天历史回溯 · BGM / CG 播放
